
Lv 0 : API 명세서 작성
| 기능명     | Method | URL                  | 요청 Body 예시                  | 응답 Body 예시                  | 응답 코드       | 비고             |
|------------|--------|-----------------------|----------------------------------|----------------------------------|------------------|------------------|
| 일정 등록  | POST   | /api/schedule         | `{ "title": "스터디", "startDate": "...", ... }` | `{ "id": 1, "message": "등록 완료" }` | 201 Created      | 일정 생성        |

| 일정 수정  | PUT    | /api/schedule/{id}    | `{ "title": "수정된 제목", ... }`| `{ "message": "수정 완료" }`    | 200 OK           | 전체 수정        |
| 일정 삭제  | DELETE | /api/schedule/{id}    | 없음                             | 없음                             | 204 No Content   | 일정 삭제        |
