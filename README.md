# 로그인
## 로그인 - 로그인 X
| 항목             | 내용                      |
|------------------|-------------------------|
| **Method**       | `POST`                  |
| **URL**          | `/login`                |
| **설명**         | 새로운 회원을 등록합니다.          |
| **Request Body** | JSON (application/json) |

### 자료형
| 필드명 | 타입     | 필수 | 설명      |
|-----|----------|----|---------|
| email | String   | O  | 회원 이메일  |
| password | String   | O  | 회원 비밀번호 |

### 요청 예시
```json
{
  "email": "abc@gmail.com",
  "password": "abcd"
}
```
### 응답 예시
```json
{
  "message": "홍길동 님 로그인을 환영합니다!"
}
```
### 상태코드
| 코드  | 설명        |
|-----|-----------|
| 200 | 로그인 성공    |
| 404 | 회원 정보 없음  |
| 401 | 비밀번호 인증 실패 |
| 500 | 서버 내부 오류  |

# 회원
## 회원가입 ( 회원 생성 ) - 로그인 X
| 항목               | 내용                      |
| ---------------- | ----------------------- |
| **Method**       | `POST`                  |
| **URL**          | `/member/register`      |
| **설명**           | 새로운 회원을 등록합니다.          |
| **Request Body** | JSON (application/json) |

### 자료형
| 필드명         | 타입     | 필수 | 설명      |
| ----------- | ------ | -- | ------- |
| member_name | String | O  | 회원 이름   |
| email       | String | O  | 회원 이메일  |
| password    | String | O  | 회원 비밀번호 |


### 요청 예시
```json
{
  "member_name": "홍길동",
  "email": "hong@example.com",
  "password": "1234"
}
```
### 응답 예시
```json
{
  "id": 1,
  "member_name": "홍길동",
  "email": "hong@example.com",
  "create_at": "2025-05-23T13:30:00.000+09:00",
  "modified_at": "2025-05-23T13:30:00.000+09:00"
}

```
### 상태코드
| 코드  | 설명               |
| --- | ---------------- |
| 201 | 회원 생성 성공         |
| 400 | 유효성 실패 (형식 오류 등) |
| 500 | 서버 내부 오류         |
